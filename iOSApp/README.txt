Install CocoaPods:
CocoaPods is the dependency manager for Swift and Objective-C Cocoa projects. It has over
ten thousand libraries and can help you scale your projects elegantly.

To install CocoaPods just run the following command in your terminal.
( $sudo gem install cocoapods )


iOSApp (mySearch):	
In order to run "mySearch" app you must install some external pods ("Alamofire", "SwiftyJSON")

To do this you just run the command ( $pod install ) from inside this folder.

Then open the .xcworkspace file select iPhone 4s for similation and press the RUN button
